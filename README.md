# 🧬 Project Symbiont: The "David vs Goliath" AI Agent

### 🇷🇺 Manifesto (Russian)
Этот проект — доказательство того, что для участия в гонке ИИ не нужны миллиардные бюджеты и дата-центры размером с город.

**Концепция:** Вместо обучения гигантской "всезнающей" модели (что дорого и неэффективно), мы создали **Агента-Симбионта**.
Он состоит из:
1.  **Tiny LLM (Qwen 1.5B)** — "карманная" нейросеть, работающая на бесплатном GPU.
2.  **Real-Time Tools** — модули доступа к Yahoo Finance и Wikipedia.

**Результат:**
Модель весом 1.5 Гб выдает **актуальную биржевую аналитику** с точностью до секунды. В то время как корпоративные гиганты (100B+ параметров) без доступа к сети галлюцинируют данными двухлетней давности.

### 🚀 Tech Stack (Стек технологий для бедных, но умных)
*   **Infrastructure:** Kaggle / Google Colab (Free Tier T4 GPU)
*   **Core Model:** `Qwen/Qwen2.5-1.5B-Instruct`
*   **Tools:** `yfinance` (Live Market Data), `wikipedia` (Knowledge Graph)
*   **Framework:** Native Python (No expensive enterprise APIs)

### ⚡ How it works (Архитектура)
1.  **Input:** Пользователь спрашивает про компанию (напр. "Nvidia").
2.  **Retrieval:** Агент идет на биржу (Yahoo) за ценой и в энциклопедию за фактами.
3.  **Fusion:** Данные смешиваются в промпте.
4.  **Inference:** Маленькая модель анализирует *контекст* и выдает экспертный ответ.
